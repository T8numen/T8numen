## Focus

- Vibe coding：从真实需求出发，用 AI 辅助把想法快速变成可运行、可维护的小项目。
- Android 折腾：Activity 启动入口、安装器体验、应用使用记录、系统能力边界和快捷操作。
- LSPosed / Xposed：围绕系统截图、权限边界、应用行为拦截做可验证的小实验。
- 自动化：用脚本、GitHub Actions 和邮件通知处理定时监控、信息整理和重复维护任务。
- Web 小工具：把简单但高频的编辑、查询、展示流程做成可直接使用的页面。
- 中文化 / 改造：让已有软件更顺手，也更适合自己的使用习惯。
