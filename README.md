## Focus

- Vibe coding：从真实需求出发，用 AI 辅助把想法快速变成可运行的项目。
- Android 折腾：Activity、安装器、应用使用记录、系统入口和快捷操作。
- 小工具：把简单但高频的编辑、查询、展示流程做成可直接使用的页面。
- 中文化 / 改造：让已有软件更顺手，也更适合自己的使用习惯。
