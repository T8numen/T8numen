## Focus

- Android 工具：Activity 管理、应用使用记录、系统入口和快捷操作。
- 自动化：用脚本或 GitHub Actions 处理重复性构建、监控和发布工作。
- Web 小工具：把简单但高频的编辑、查询、展示流程做成可直接使用的页面。
- 二次改造：对已有项目做中文化、体验整理或维护性修补。
